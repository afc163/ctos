# C2S

c2s is a tool to transform component package to [spm@3x](http://www.spmjs.io/).

## Install

```bash
$ npm install c2s -g
```

## Usage

```bash
# transform specific version
$ c2s visionmedia/page.js@1.3.4

# transform all versions under a repo
$ c2s visionmedia/page.js
```
