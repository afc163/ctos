# Component Transformer

Component transformer is a tool to transform component package to spm@3x package.

## Install

```bash
$ npm install c2s -g
```

## Usage

```bash
# install specific version
$ c2s visionmedia/page.js@1.3.4

# install all versions
$ c2s visionmedia/page.js
```
