# ctos

a tool to transform component package to [spm@3x](http://www.spmjs.io/).

## Install

```bash
$ npm install ctos -g
```

## Usage

```bash
# transform specific version
$ ctos visionmedia/page.js@1.3.4

# transform all versions under a repo
$ ctos visionmedia/page.js
```
